How it Works ?

  • This software runs simply based on the copy paste mechanism adopted by the people in the past.
  • A separate browser for your account and a separate account for your friend.
  • Firstly it runs through each and every question from both the given accounts at a given time and after getting all the questions it compares it with your account and the paste it and Evaluates it (All by the help of PYTHON).

Prerequisite

  • OS : Windows (any)
  • Browser : Google Chrome
  • Chrome Extension : Don't fuck with paste
  • Python 3+
  • Patience

Instruction's

  • Extract the .zip file
  • Double Click the Elab.exe Executable (It will take some time)
  • Fill the Details
    • Registration - your's
    • Password - your's
    • Select Campus
    • Select Language
    • Friend's credentials
      • In Given Format :
          • Registration_No:Password {space} Registration_No:Password {space} etc

          Ex - [RA1711003040134:Shiva123 RA1711003040134:Shiva123]

  • Click the 'Mouse' Button and select an area anywhere inbetween the Specified part (RED BOX) and press "CTRL + SHIFT + X"
  • Press the 'RUN' button
  • Once the browser's popup don't use the mouse
  • Sit back and Enjoy
